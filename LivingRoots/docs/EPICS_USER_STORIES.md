# Living Roots Project Epics and User Stories

This document provides a detailed breakdown of all epics and their corresponding user stories created for the Living Roots project.

## Early Game Epics (Year 1)

### Epic 1: Soil Health (Core) - Issue #1
- **Description**: Core system for tracking and managing soil health across the farm
- **User Stories**:
  - US-01-01: Save and Load Soil Health Values (Issue #22)
  - US-01-02: Visualize Soil Health (Issue #23)
  - US-01-03: Soil Health Degradation (Issue #24)
  - US-01-04: Soil Health Improvement with Compost (Issue #25)

### Epic 2: Composting - Issue #2
- **Description**: System for creating and using organic compost
- **User Stories**:
  - US-02-01: Create Composter Machine (Issue #26)
  - US-02-02: Create Organic Compost Item (Issue #27)

### Epic 3: Simple Polyculture - Issue #3
- **Description**: Implementing companion planting with "Three Sisters"
- **User Stories**:
  - US-03-01: Detect Three Sisters Configuration (Issue #28)
  - US-03-02: Implement Three Sisters Bonuses (Issue #29)

### Epic 4: Soil Cover (Mulch) - Issue #6
- **Description**: Creating mulch to prevent soil degradation
- **User Stories**:
  - US-04-01: Create Mulch Item (Issue #30)
  - US-04-02: Implement Mulch Application Effects (Issue #31)

### Epic 5: New NPC - Elena - Issue #7
- **Description**: Adding Elena NPC as farming mentor
- **User Stories**:
  - US-05-01: Create Elena NPC Character (Issue #32)
  - US-05-02: Implement Elena's Agroecology Guidance (Issue #33)

### Epic 6: Heirloom Seeds - Issue #8
- **Description**: Introducing sustainable heirloom seeds
- **User Stories**:
  - US-06-01: Implement Heirloom Seed Items (Issue #34)
  - US-06-02: Differentiate Heirloom Effects from Regular Seeds (Issue #35)

### Epic 7: Rainwater Harvesting - Issue #9
- **Description**: Collecting rainwater for sustainable irrigation
- **User Stories**:
  - US-07-01: Create Small Cistern Crafting Recipe (Issue #36)
  - US-07-02: Implement Rainwater Collection Mechanics (Issue #37)
